TLI Builder

Plan your perfect build

TLI Builder is a free, open-source build planner for Torchlight: Infinite. Theory-craft talent trees, configure slates, equip legendary gear, and preview your full stat sheet - all before you spend a single point in-game.

Features
🌳
Talent Trees
Browse and allocate nodes across every talent tree. Search nodes by effect text and preview stat scaling at each rank.
🔮
Slates
Configure Creator Slates and their node selections. Slate bonuses feed directly into the live stat sheet.
⚔️
Legendary Gear
Browse the full legendary catalog, customize affix values with sliders, and equip items into named gear slots.
📊
Live Stat Sheet
See your complete stat breakdown update in real time. Hover any stat to see exactly which talents, slates, and gear contribute to it.
💾
Build Saving
Save and load multiple named builds locally. Switch between them instantly without losing your work.
🆓
Free & Open Source
TLI Builder is completely free to use. The full source is available on GitHub - contributions welcome.
